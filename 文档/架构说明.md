# c2rust-translate 架构说明

## 项目结构

```
src/
├── lib.rs              - 主工作流程编排
├── common_tasks.rs     - 公共任务
├── initialization.rs   - 项目初始化
├── verification.rs     - 构建验证和修复
├── builder.rs          - Cargo 构建
├── translator.rs       - C 到 Rust 翻译
├── analyzer.rs         - 代码分析
├── interaction.rs      - 用户交互
├── file_scanner.rs     - 文件扫描
├── git.rs              - Git 版本控制
└── ...
```

## 核心模块说明

### 1. common_tasks 模块（公共任务）

提供4个标准化的公共任务：

1. **execute_code_error_check**: 执行代码错误检查
2. **execute_code_warning_check**: 执行代码告警检查
3. **execute_hybrid_build_check**: 执行混合构建检查
4. **execute_translation_task**: 执行翻译任务

### 2. 主工作流程

```
1. 初始化 feature 目录
2. 执行初始验证（代码错误检查）
3. 扫描待翻译文件
4. 进入翻译循环
   4.1 执行翻译（失败 → 记录 TranslationFailedSignal，跳过该文件）
   4.2 执行代码错误检查（带修复循环；单次修复失败 → 警告并继续）
   4.3 执行代码告警检查（带修复循环；单次修复失败 → 警告并继续）
   4.4 git 提交（失败 → 警告并继续，工作树可能保持脏状态）
5. 打印统计信息
```

### 3. 错误信号类型（`verification.rs`）

| 类型 | 用途 |
|------|------|
| `SkipFileSignal` | 用户手动选择跳过，或 `C2RUST_AUTO_RETRY_ON_MAX_FIX` 触发的自动跳过。文件记录到 `TranslationStats::skipped_files`，事后可通过 `handle_skipped_files_loop` 再次处理 |
| `TranslationFailedSignal` | 翻译命令本身失败（脚本非零退出）。文件记录到 `TranslationStats::translation_failed_files`，与 `skipped_files` 分开统计，不在 retry 循环中再次出现 |

### 4. 错误处理策略

工具采用"尽力而为"策略，只有 `code_analyse` 失败是致命的：

| 失败类型 | 处理方式 | 是否致命 |
|----------|----------|---------|
| 翻译失败 | 记录到 `TranslationStats::translation_failed_files`（与 `skipped_files` 分开），跳过该文件继续下一个，最终统计单独显示 | 否 |
| 修复步骤失败 | 打印警告，本次修复计数为 0，循环继续 | 否 |
| git 提交失败 | `git_commit_or_warn` 打印警告并返回 `false`，工作树可能脏 | 否 |
| `code_analyse` 失败 | `?` 传播，立即中止流程 | **是** |

## 命名规范

- `execute_*`: 执行某项任务
- `*_with_fix_loop`: 带自动修复循环的检查
- `*_or_warn`: 失败时打印警告并继续（非致命），如 `git_commit_or_warn`
- 避免使用 `gate_` 前缀（已废弃）

## 环境变量

| 变量名 | 说明 |
|--------|------|
| `C2RUST_PROCESS_WARNINGS` | 设为 `0` 或 `false` 可跳过 Phase 2（警告检测与修复），默认启用 |
| `C2RUST_TEST_CONTINUE_ON_ERROR` | 设为 `1`、`true` 或 `yes`（大小写不敏感）时，`c2rust_test` 失败不会中断流程，仅记录警告；默认禁用（测试失败为致命错误） |
| `C2RUST_AUTO_RETRY_ON_MAX_FIX` | 设为 `1`、`true` 或 `yes`（大小写不敏感）时，修复次数达上限后自动重试翻译，保证全自动运行；最后一次翻译机会耗尽时自动跳过文件；默认禁用（显示交互式提示） |

## 公共任务调用示例

```rust
// 在任何需要检查的地方调用
crate::common_tasks::execute_code_error_check(feature, show_full_output)?;
crate::common_tasks::execute_code_warning_check(feature, show_full_output)?;
```
